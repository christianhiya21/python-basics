# python-basics
A beginner-friendly Python project containing basic programs to practice core concepts: loops, conditionals, lists, and user input. Includes a Guessing Game, Calculator, To-Do List, and Even/Odd Checker.
# Python Basics

This repository contains beginner-friendly Python programs for practice.  
All programs are collected in a single file `Python_Basics.py`.

## Programs Included

1. **Guessing Game**  
   - A number guessing game where the user tries to guess a random number.  
   - Teaches: `while` loops, conditionals, user input, and randomness.

2. **Calculator**  
   - A simple calculator that supports addition, subtraction, multiplication, and division.  
   - Teaches: `if/elif` statements, user input, type conversion.

3. **To-Do List**  
   - A basic interactive to-do list to add and remove tasks.  
   - Teaches: `lists`, `while` loops, `for` loops, `enumerate()`.

4. **Even/Odd Checker**  
   - Checks whether a number entered by the user is even or odd.  
   - Teaches: conditionals, modulus operator `%`, user input.

## How to Run

1. Download `Python_Basics.py` from this repository.  
2. Open it in a Python environment (Python 3 recommended).  
3. Run the file and follow the instructions in the console.
